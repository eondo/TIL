## Docker

#### 볼륨이란?

> 호스트 머신의 폴터를 도커 컨테이너 내부의 폴더에 매핑하는 것

- 도커 컨테이너가 내려가도 호스트 머신의 폴더는 내려가지 않게 된다.
- 볼륨 매핑을 해두면 마운트 된 상태에서는 공유 폴더처럼 사용이 가능하다.

#### 도커 파일 명령어
- `FROM` : 이미지를 가져옴
  - java 기반 프로젝트를 빌드/실행하려면 꼭 필요함
- `ARG` : 빌드 변수 설정
  - JAR_FILE이라는 변수명에 build/libs 안의 .jar 파일을 할당
- `COPY` : 파일 복사
